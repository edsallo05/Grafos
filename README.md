Grafos en Java - BFS y DFS

Este proyecto implementa recorridos de grafos utilizando:
	•	Lista de adyacencia (grafo dirigido con peso)
	•	Matriz de adyacencia (grafo no dirigido sin peso)

📌 Contenido

1. Grafo con lista de adyacencia
	•	Representación con Map y LinkedList
	•	Aristas con peso
	•	Implementación de:
	•	BFS (Breadth-First Search)
	•	DFS (Depth-First Search)

2. Grafo con matriz de adyacencia
	•	Representación con matriz int[][]
	•	Grafo no dirigido
	•	Implementación de:
	•	BFS (orden ascendente por etiqueta)
	•	DFS (siguiendo el orden de columnas)

🧠 Conceptos aplicados
	•	Grafos dirigidos y no dirigidos
	•	Listas de adyacencia
	•	Matrices de adyacencia
	•	Recorridos BFS y DFS
	•	Uso de colas y pilas
